Module Introduction

This oscilloscope is a dedicated digital oscilloscopefor professional vehicle maintenance and test.

Main Technical Parameters:

          Supply Voltage: 12V DC

          Working Temperature: -30℃~+50℃

Oscilloscope Performance Parameters:

          Channel Numbers: 4 channels

          Sampling Frequency: 100MHz

          Sampling Accuracy: 8 digits

          Voltage Range: 10mV-5V/div

          Scanning Time: 5nS-1000S/div 

 

Function Introduction:

       <1> Provide 4-channel collection input function. Each channel can separately set timebase, voltage, phase and coupled mode, etc.

       <2> Provide 3 trigger methods: Automatic trigger, normal trigger and single trigger;

       <3> Provide wave form time and voltage measurement function;

       <4> Provide 2 trigger modes: Edge trigger and pulse trigger;

       <5> Provide automatic correction function;

       <6> Provide expert setting function;

       <7> Provide wave picture storage, wave form storage and wave form record function;

       <8> Provide wave form playback function;

       <9> Provide wave form automatic cache function;
