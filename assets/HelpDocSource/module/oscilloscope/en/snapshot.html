File

In Interface Guide introduced D area, find [File], click it to enter menu option of file management .

File Function Bar Introduction

Enter menu option of file management (above image)

[Save Snapshot]: When oscilloscope signal input suspended, it saves current oscilloscope collecting status as snapshot.

[Snapshot Management]: Stored snapshot can be managed via Snapshot Management.

[Wave Form Record]: Under the status of oscilloscope signal input, it records current data wave form collected by oscilloscrope.

[Wave Form Playback]: Recorded wave form can be played back via this Wave Form Playback.

Save Snapshot

Click [Run/Pause] button, suspend oscilloscope signal input to save current status as snapshot. Click [Sava snapshot] button to save.

Click [Save Snapshot] button to enter snapshot information edit interface, see below image. Separately edit the content marked in below image 1, 2, 3, complete snapshot storage.    

Snapshot Management

Click [File]->[Snapshot Manager] button to enter snapshot management interface.

Click [Edit] button to enter snapshot information edit interface, refer to the edit of [Save Snapshot].

Wave Form Record

Click [File]->[Wave Form Record] button to enter wave form record interface. Record wave form according to the interface prompt.Totally you can record 1,000 pages.if you do not want to record so many pages, you can stop and sava current recorded pages by clicking [Stop] button.

Click [File]->[Wave Form Playback] button to enter wave form playback interface. Select wave form data to be played back. See below image.
