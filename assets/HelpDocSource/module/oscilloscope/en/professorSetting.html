Expert

         In Interface Guide introduced D area, find [Expert], click it to enter expert setting interface. Expert setting pre-stores standard oscilloscope parameters measured by each sensor. Select corresponding expert wave form and confirm, oscilloscope will preset according to the preset value of the sensor.

Above image is the interface of [Expert] setting.

A：Listed expert files;

B：Listed basic channel parameters of selected expert setting file.

C：Status display area, displaying every working parameter and status of oscilloscrope;

D:Ref channel selection;

E：Formula bar of expert setting file.

File Edit

Use edit function button on formula bar to edit expert setting file.

[Refresh]: Refresh interface;

[Delete]: Delete selected expert setting file;

[Edit]: Access edit interface of expert setting file to edit selected file. See below image;

[Confirm]: Set the selected expert setting file to current oscilloscope setting;

[Return]: Return to oscilloscope page.

Note: Deselect the options in<Add to Expert Library>to remove the option from expert library.

Base Reference

   InInterface Guide introduced D area, find [Base Reference], click it to enter base reference interface. Base reference pre-stores standard oscilloscope wave data of some base sensors.
