Parameter Setting

After access setting, we can use Interface Guide[Automatic Setting] in D area to set parameters automatically. We can also do individual setting to every basic parameter of oscilloscrope.

Horizontal Setting

After access setting, we can use Interface Guide introduced E area to find horizontal setting.

Horizontal Setting Introduction

<1> It sets the timebase in oscilloscope display area;

<2> T-Y format includes normal, scanning and rolling.

     Normal: When memory is full with collected data, data will be displayed;

     Scanning: Continuously trigger. Data will be displayed from left to right when it meets the trigger condition;

     Rolling: Continuously display real time data from right to left.

 

Vertical Setting

After access setting, we can use Interface Guide introduced F area to find vertical setting.

Vertical Setting Introduction

<1> Channel Setting: 4 channels CH1-CH4 are available;

<2> It sets the voltage (V/L) in oscilloscope display area;

<3> Channel coupling method includes direct current, alternate current and grounding;

<4> Detector Setting: It amplifies the signal, maximum 1000 times;

<5> Channel Phase Selection: Tick the choice box to set channel phase as reverse direction. Without tick, it is forward direction.

 

Trigger Setting

After access setting, we can use Interface Guide introduced G area to find trigger setting.

Trigger Setting Introduction

<1> Trigger mode includes pulse width trigger and edge trigger.

Pulse Width Trigger: Collect channel data according to pulse width condition of the channel;

Edge Trigger: Collect channel data according to edge type of the channel.

<2> Trigger method includes automatic, normal and single triggers, refer to the setting.

Automatic: Continuously trigger. When trigger conditions are satisfied, it triggers accordingly. When trigger conditions are not satisfied, it is forced to trigger.

Normal: Continuously trigger. When trigger conditions are satisfied, it triggers accordingly. When trigger conditions are not satisfied, it will not trigger.

 Single: Single trigger. When trigger conditions are satisfied, it triggers accordingly but only one time, then stops.

<3> Four signal sources CH1-CH4 can be separately selected as trigger signal source.

<4> Pulse width condition selection includes greater, less and equal. This item needs to be set only when trigger mode is pulse width trigger.

Greater: Greater than specified pulse width trigger.

Less: Less than specified pulse width trigger.

Equal: Equal to specified pulse width trigger.

<5> Pulse Setting: It sets the width of trigger pulse width. This item needs to be set only when trigger mode is pulse width trigger.

<6> Edge type includes top edge trigger and bottom edge trigger. This item needs to be set only when trigger mode is edge trigger.

<7> In trigger setting, there's an option of high frequency restraint. Tick means select the setting. No tick means not set.
