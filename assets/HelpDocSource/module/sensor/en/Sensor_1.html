Sensor

Vehicle sensor is a signal input device of vehicle electronic control system. It converts all kinds of running parameters, such as vehicle speed, water temperature, engine speed, air flow and throttle opening, etc, into electrical signals and sends to ECU, which regulates the engine running status according to these parameters to keep it in the best working condition. There are many types of car sensors. When judge sensor fault, consider not only sensor itself, but also the whole circuit of this fault and parameter changes during vehicle running. "Sensor" function is specially developed for sensor fault diagnosis. Users can easily perform diagnosis to sensor. Below is the main interface of "Sensor":

DC voltage analog

Fixed frequency analog

Predefined wave form analog

Hand-drawn wave form analog

Select one of the functions to enter corresponding analog function.
