Battery Performance Standard

Vehicle battery status mainly includes:

1. Good Battery

2. Replace

3. Good, Recharge

4. Recharge, Retest

5. Bad Battery

The specified capacity range of battery test varies based on different test standard, details as below:

CCA: 100--1700

DIN: 100--1200

EN: 100--1700

IEC: 100--1200

GB: 100--1200

SAE: 100--1700

JIS: 26A17--245H52

Default capacity range of unspecified test standard is 100-1700.
